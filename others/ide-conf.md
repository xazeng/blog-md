常用软件的配置和快捷键需要统一规划下，以便于记忆。
<!--more-->

---------------------------------------------------------

## visual studio

### 快捷键 Tools->Options->Environment->Keyboard

* 切换 .h 和 .cpp  
  Alt+O -> ToggleHeaderCodeFile

* 查找文件  
  Shift+Alt+O -> GoToFile

* 工程中搜索  
  Ctrl+;

* 查找成员变量  
  Alt+M -> GoToMember

---------------------------------------------------------

## vs code

### 快捷键 File->Preferences->Keyboard Shortcuts

* 切换 .h 和 .cpp  
  Alt+O

* 查找文件  
  Shift+P

* 查找成员变量  
  Ctrl+Shift+O
